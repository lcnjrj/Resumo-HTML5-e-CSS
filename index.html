É uma parte fundamental do documento HTML que define o idioma principal utilizado no conteúdo da página. Nesse caso, "en" é o código ISO 639-1 para o idioma inglês.

<head>
A tag é uma parte fundamental de um documento HTML. Ela fornece informações sobre o documento, como metadados, configurações, scripts, estilos e referências a recursos externos. A tag não contém o conteúdo visível na página, mas sim informações sobre como a página deve ser processada e exibida.

<meta charset="UTF-8"> É uma tag HTML que define o conjunto de caracteres usado pelo documento. Nesse caso específico, define o conjunto de caracteres como UTF-8.

<meta http-equiv="X-UA-Compatible" content="IE=edge"> A tag com o atributo http-equiv="X-UA-Compatible" é uma instrução específica para o Internet Explorer (IE) sobre o modo de renderização a ser usado ao exibir uma página da web. http-equiv="X-UA-Compatible" é um atributo que especifica que a meta tag está sendo usada para fornecer informações de cabeçalho HTTP, em vez de metadados convencionais. content="IE=edge" indica ao Internet Explorer que a página deve ser renderizada usando a versão mais recente do mecanismo de renderização disponível. Neste caso, "edge" significa usar o modo de exibição mais atual disponível no Internet Explorer, ignorando possíveis modos de compatibilidade ou versões mais antigas do mecanismo de renderização do IE.
